1. All your requests will go to **api.lalals.com**
2. Lalals API offers 6 different API features: 

| Feature | Description                           |
| -------- | ------------------------------------- |
| Music AI     | Promt to Song|
| Music AI + Voice      | Promt + Selected Voice to Song                        |
| Voice Changer   | Change any Vocal to any Voice |
| Cover   | Create a Cover song |
| Text To Speech   | Selected Voice will say anything |
| Vocal/Instrumental   | Extract Vocals or Instrumentals |

#### Lets generate a country song about our Love towards AI Generated Music ‚ù§Ô∏èü§†

3. For this we will use the Music AI Feature
```
curl -X 'POST' \ 
'https://api.lalals.com/api/public/v1/MusicAI' \
-H 'accept: application/json' \
-H 'Authorization: <api_key>' \
-H 'Content-Type: application/json' \
-d '{
"music_style": "Song about my love to AI Music in Country Style",
"webhook_url": "http://api.lalals.com",
}'
```
